# Proyecto Final de Automatización de Procesos con IA

Este repositorio contiene una propuesta ejecutiva para automatizar procesos en un concesionario BMW usando IA, Zoho One, WhatsApp Business API, integración web y conexión con ERP/Taller actual.

## Objetivo

Centralizar datos, evitar citas duplicadas, mejorar la atención al cliente y automatizar procesos de taller, call center, ventas, administración y recursos humanos.
